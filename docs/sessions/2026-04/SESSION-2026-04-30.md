## Lições aprendidas (para próximas sessões)

1. **`bun add` pode não persistir no `package.json`** se interrompido. Sempre conferir após adicionar dep crítica.
2. **Vercel prefere `npm` se houver `package-lock.json`** — para projetos Bun, ou apagar o lock-npm ou usar `vercel.json` com `installCommand` explícito.
3. **TanStack Query v5+ passa 2º argumento em `mutationFn`** — testes que asseram chamadas precisam de `expect.anything()` no 2º arg.
4. **Vercel Deployment Protection** vem ativa por padrão em deploys com hash; URL principal sem hash é pública.
